# ☕ [자기야, 카페 어디갈래?] 
### 카테고리 기반 카페 추천 서비스입니다

## ❓ Motivation
#### 전국 팔도에 다양한 카페는 많다. 
#### 하지만 카페에 가는 이유는 전부 다르다.
#### 나는 공부하려고 하는데 여기는 들어가도 되는걸까...?
#### 공부하러 온 사람들. 수다떨러 온 사람들. 서로가 눈치보는게 아닐까?

#### 카테고리 기반 카페 추천 서비스, "자기야 카페 어디갈래" 입니다.
