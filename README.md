## ❓ Motivation
#### 전국 팔도에 다양한 카페는 많다. 
#### 하지만 카페에 가는 이유는 전부 다르다.
#### 나는 공부하러 가고 싶은데 여기는 들어가도 되는걸까...?
#### 공부하러 온 사람들. 수다떨러 온 사람들. 서로가 눈치보는게 아닐까?
